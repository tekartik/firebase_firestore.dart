# firestore_sembast

Sembast simulation for unit test, flutter, vm and browser

## Setup

In `pubspec.yaml`:
```yaml
  tekartik_firebase_firestore_sembast:
    git:
      url: git://github.com/tekartik/firebase_firestore.dart
      path: firestore_sembast
      ref: null_safety
    version: '>=0.8.0'
```
